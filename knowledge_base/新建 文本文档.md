# 西安交通大学“西迁”历史事件知识库

## 1. 核心概念与关键术语

- **西迁 (Xī qiān)**：特指1956年至1957年间，响应国家号召，交通大学主体部分（包括师生、设备、档案等）从上海搬迁至陕西西安的历史事件。这是中国高等教育布局的一次重大战略调整。
    - **易错纠正**: "西迁" (Xī qiān) 是专有名词，指“向西迁移”，易被误转录为同音的“系迁” (xì qiān)、“席迁”等。
- **交通大学西迁 (Jiāotōng Dàxué Xī qiān)**：同“西迁”，是该事件的完整表述。
- **交大西迁 (Jiāo Dà Xī qiān)**：常用简称。
- **西迁精神 (Xī qiān Jīngshén)**：指在“西迁”过程中形成的，以“胸怀大局、无私奉献、弘扬传统、艰苦创业”为核心的精神品质。
- **西迁人 (Xī qiān Rén)**：指当年参与、经历“西迁”的教职员工、学生及家属。
- **西安交通大学 (Xī'ān Jiāotōng Dàxué)**：简称“西安交大”。由“西迁”的交通大学主体发展而来。
- **上海交通大学 (Shànghǎi Jiāotōng Dàxué)**：简称“上海交大”。交通大学的另一部分留在了上海，发展为现在的上海交通大学。两校同根同源。
- **“迁校”与“分设”**：起初的决策是“迁校”，即整体搬迁。后续根据实际情况，中央决定“分设”，即西安和上海两地办学。但主体力量和绝大部分师生、设备均迁至西安。

## 2. 历史背景与决策过程

- **时间**: 20世纪50年代中期，新中国“一五计划”期间。
- **背景**:
    1.  **国家战略需求**：为了平衡全国高等教育布局，支持西北地区（特别是以西安为中心的工业基地）的建设发展，需要一所高水平的理工科大学。
    2.  **国际形势**：当时国际局势紧张，国家考虑将重要科研和教育资源从沿海向内地疏散，以备战备荒。
- **决策**:
    - **1955年**：中华人民共和国国务院（时任总理周恩来）决定将交通大学内迁西安。
    - **1956年**：交通大学开始大规模西迁。
    - **1957年**：大部分师生完成迁移。
    - **1959年**：国务院正式批准交通大学西安部分和上海部分独立建校，分别命名为“西安交通大学”和“上海交通大学”。

## 3. “西迁”过程概述

- **动员与响应**：时任交通大学校长彭康同志在全校师生中进行了深入的思想动员。广大师生（特别是许多老教授、老专家）响应国家号召，克服个人和家庭困难，自愿报名西迁。
- **迁移规模**：
    - **人员**：数千名教职员工（包括教授、讲师、行政人员、工人）和近万名学生。
    * **物资**：数万吨的教学设备、科研仪器、图书资料、档案等，几乎是“连根拔起”。
- **西迁专列**：满载师生和物资的火车专列从上海出发，开往西安。
- **艰苦创业**：到达西安初期，办学条件和生活条件极其艰苦。校园尚在建设中（许多地方还是麦田），师生们“一边建校，一边办学”，住在简易的宿舍，在临时搭建的教室里上课。

## 4. 关键人物（部分）

（注意：访谈中提及的老教授名字极易转录错误，此列表可用于RAG纠错）

- **彭康 (Péng Kāng)**：时任交通大学校长兼党委书记，是“西迁”的主要领导者和组织者。
- **陈学俊 (Chén Xuéjùn)**：中国科学院院士，热能工程学家，“西迁”时积极响应号召的著名教授之一。
- **胡奈赛 (Hú Nàisài)**：著名内科专家、医学教育家，原上海第二医学院（交大医学院前身之一）教授，随迁西安。
- **钟兆琳 (Zhōng Zhàolín)**：中国电机之父，著名电机工程专家，不顾年迈积极带头西迁。
- **陆定一 (Lù Dìngyī)**：时任中共中央宣传部部长，曾到校动员。
- **周恩来 (Zhōu ēnlái)**：时任国务院总理，是“西迁”的最高决策者之一。
- （访谈中可能还会提及其他具体的老教授姓名，如：**沈尚贤 (Shěn Shàngxián)**, **赵富鑫 (Zhào Fùxīn)**, **陈大燮 (Chén Dàxiè)**, **张鸿 (Zhāng Hóng)** 等“西迁”代表人物。）

## 5. “西迁精神”的详细阐释

（这些是访谈中可能高频出现的短语，易被误识）

- **胸怀大局 (Xiōnghuái Dàjú)**：指“西迁”师生们以国家发展大局为重，个人利益服从国家利益。
- **无私奉献 (Wúsī Fèngxiàn)**：指他们放弃上海优越的生活条件，到艰苦的西北开创新事业。
- **弘扬传统 (Hóngyáng Chuántǒng)**：指继承和发扬交通大学“精勤求学、敦笃励志、果毅力行、忠恕任事”的校训和“起点高、基础厚、要求严、重实践”的办学传统。
- **艰苦创业 (Jiānkǔ Chuàngyè)**：指在西安白手起家，克服重重困难建设新校园和新学科。
- **“党让我们去哪里，我们背上行囊就去哪里”**：这是“西迁”师生常说的一句话，体现了他们的决心和服从精神。

## 6. 地点与设施

- **上海 (Shànghǎi)**：交通大学的迁出地。
- **西安 (Xī'ān)**：交通大学的迁入地。
- **兴庆宫公园 (Xīngqìnggōng Gōngyuán)**：西安交通大学兴庆校区所在地，原址是唐代兴庆宫遗址的一部分。
- **“梧桐树”**：从上海迁移过来的梧桐树苗，栽种在西安交大校园内，是“西迁”的象征之一。
- **“饮水思源碑” (Yǐnshuǐ Sīyuán Bēi)**：交通大学的标志性纪念物，上海和西安两地校园均有。
